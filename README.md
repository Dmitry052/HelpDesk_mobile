### For android
`````
- yarn install
- set your 'serverUrl' to config.js
- move to 'android/app/' your google-services.json
- react-native run-android
`````
### For ios
`````
- yarn install
- set your 'serverUrl' to config.js
- move to 'ios/mobile/' your GoogleServices-info.plist
- react-native run-ios --simulator="iPhone {model}"
`````
